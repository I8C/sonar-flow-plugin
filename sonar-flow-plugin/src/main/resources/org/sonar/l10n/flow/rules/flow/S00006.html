Services with interface element 'EXIT' need to be configured correctly. 

  Services with interface element 'EXIT' step need to be configured correctly:
  	- The "Exit from " property needs to be configured.
  	- The failure message needs to be added if the "Signal" property is set to "Failure".

Noncompliant Code Example

Any non-compliant variant

Compliant Solution

	- Configure the "Exit from " property.
  	- Add the failure message if the "Signal" property is set to "Failure".
